
## Инструкция по развертыванию и запуску


### Клонирование репозитория
Откройте терминал на вашем компьютере, перейдите в папку, куда хотите скачать проект, и выполните команду:
```bash
git clone https://github.com/anapty-xi/exam
```


### Создание и активация виртуального окружения
Чтобы изолировать библиотеки проекта от вашей операционной системы, создайте виртуальное окружение `venv`:

* **Для Windows:**
  ```bash
  python -m venv venv
  venv\Scripts\activate
  ```
* **Для Linux / macOS:**
  ```bash
  python3 -m venv venv
  source venv/bin/activate
  ```
*После активации в начале строки терминала должна появиться плашка `(venv)`.*

### Установка зависимостей
Установите все необходимые библиотеки (Flask, SQLAlchemy, PyMySQL) одной командой из файла `requirements.txt`:
```bash
pip install -r requirements.txt
```

###  Запуск веб-сервиса
Запустите главный файл приложения:
```bash
python main.py
```
*При первом старте система автоматически создаст нужные таблицы в MySQL и добавит стандартного администратора.*

##  Структура проекта
* `main.py` — Точка входа, создание экземпляра Flask-приложения и инициализация БД.
* `models.py` — Конфигурация подключения к XAMPP и декларативное описание модели `User` на SQLAlchemy 2.0.
* `services.py` — Сервисный слой, инкапсулирующий чистую бизнес-логику и SQL-запросы (`select`, `where`).
* `routes.py` — Веб-контроллеры (Blueprint), отвечающие за обработку HTTP-запросов и сессий.
* `templates/` — HTML-шаблоны страниц (`login.html`, `admin.html`, `success.html`).
